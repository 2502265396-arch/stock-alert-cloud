# A股云端扫描系统

每天 3 次自动扫描全市场 A 股，经过严格保守的量化+K线技术双重过滤，发现候选股时通过钉钉推送。

## 工作原理

```
全市场 5000+ 只股票
  → Layer 1 量化初筛 (涨幅/成交额/PE/主力流入/市值)
  → Layer 2 K线技术过滤 (MACD/RSI/均线/量比)
  → 钉钉机器人推送
```

## 部署

1. Fork 本仓库
2. 在 Settings → Secrets → Actions 中添加:
   - `DINGTALK_WEBHOOK`: 你的钉钉机器人 Webhook 地址
3. GitHub Actions 会自动按工作日 8:57/10:03/14:07 运行

## 本地运行

```bash
pip install -r requirements.txt
python stock_screen.py
```

## 扫描标准 (严格保守型)

| 条件 | Layer 1 | Layer 2 |
|------|---------|---------|
| 涨幅 | 2-18% (区分涨跌停板) | — |
| 成交额 | > 1亿 | — |
| PE | 10-50 | — |
| 市值 | > 50亿 | — |
| 主力净流入 | > 0 | — |
| MACD | — | 主线>0 且 柱>0 |
| RSI | — | 40-70 |
| 均线 | — | 价格 > 50日SMA |
| 量比 | — | > 0.8 |

> ⚠️ 本系统仅供研究参考，不构成投资建议。
